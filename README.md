# webpack demo

#### 创建工程
- npm init 一直回车
- npm install webpack --save-dev
- mkdir src
- mkdir dist

#### plugins
- ```npm install html-webpack-plugin --save-dev```

#### 打包步骤

- **webpack.config.js**使用`webpack`
- **webpack.dev.config.js**使用`webpack --config webpack.dev.config.js`
